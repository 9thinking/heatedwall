# 液滴撞击加热壁面仿真 - 快速入门指南

## 项目概述

本项目使用 Basilisk 软件实现液滴撞击加热固体表面的数值仿真。模拟包含两相流动、表面张力效应,并提供定量和定性分析结果。

## 主要特点

✓ **完整的物理模型**:包含两相流、表面张力、自适应网格加密
✓ **定量输出**:铺展因子、接触时间、动能演化、质心轨迹
✓ **定性输出**:界面形状演化、速度场、压力场、可视化动画
✓ **灵活的参数设置**:支持不同液滴尺寸、速度、流体性质

## 快速开始

### 1. 编译仿真程序

```bash
make
```

### 2. 运行仿真

```bash
# 使用默认参数运行
./droplet_impact

# 或使用 make 命令
make run
```

### 3. 查看结果

仿真完成后会生成以下文件:

**定量数据** (`quantitative_output.dat`):
- 时间历程数据
- 铺展因子 (D/D₀)
- 质心位置
- 体积和动能

**定性数据**:
- `interface-*.dat`: 界面形状快照
- `field-*.dat`: 完整流场数据
- `movie.mp4`: 动画视频

## 仿真参数

### 默认物理参数 (类水液滴)

- 液滴直径: D₀ = 2 mm
- 撞击速度: U₀ = 1.0 m/s
- 液体密度: ρₗ = 1000 kg/m³
- 液体粘度: μₗ = 1×10⁻³ Pa·s
- 表面张力: σ = 0.072 N/m
- 液滴温度: T = 20°C
- 壁面温度: T_wall = 100°C

### 无量纲数

程序自动计算:
- **雷诺数** Re = ρₗU₀D₀/μₗ (惯性力/粘性力)
- **韦伯数** We = ρₗU₀²D₀/σ (惯性力/表面张力)
- **奥内佐格数** Oh = √We/Re (粘性/惯性-毛细)

## 自定义仿真

### 改变撞击速度

```bash
# 快速撞击 (2 m/s)
./droplet_impact 9 11 2.0

# 慢速撞击 (0.5 m/s)
./droplet_impact 9 11 0.5
```

### 改变网格分辨率

```bash
# 高分辨率 (更精确,更慢)
./droplet_impact 10 12

# 低分辨率 (快速测试)
./droplet_impact 7 9
```

### 修改物理属性

编辑 `droplet_impact.c` 中的参数:

```c
// 液滴属性
double D0 = 2e-3;        // 直径 [m]
double U0 = 1.0;         // 速度 [m/s]
double RHO_L = 1000.0;   // 液体密度 [kg/m³]
double MU_L = 1e-3;      // 液体粘度 [Pa·s]
double SIGMA = 0.072;    // 表面张力 [N/m]
```

然后重新编译:
```bash
make clean
make
```

## 典型仿真场景

### 场景 1: 温和撞击 (We < 10)
```bash
./droplet_impact 9 11 0.5
```
预期: 铺展但不飞溅,表面张力主导

### 场景 2: 标准撞击 (10 < We < 30)
```bash
./droplet_impact 9 11 1.0
```
预期: 明显铺展,可能形成冠状结构

### 场景 3: 高速撞击 (We > 30)
```bash
./droplet_impact 9 11 2.0
```
预期: 飞溅,可能产生二次液滴

## 关键输出指标

### 铺展因子 (D/D₀)
- 液滴在壁面上的瞬时直径与初始直径之比
- 反映铺展程度
- 典型值: 1.5 - 4.0

### 接触时间
- 液滴与表面接触的持续时间
- 重要的工程参数
- 影响热量和质量传递

### 动能演化
- 反映能量耗散过程
- 粘性耗散和表面能转化

## 物理意义

### 铺展阶段
- 动能转化为表面能和粘性耗散
- 液滴径向扩展

### 最大铺展
- 动能完全转化为表面能
- 铺展达到最大值

### 回缩阶段
- 表面张力驱动液滴回缩
- 可能发生反弹

## 常见问题

### 编译错误: "qcc: command not found"
确保 Basilisk 已正确安装并设置环境变量:
```bash
export BASILISK=$HOME/basilisk/src
export PATH=$PATH:$BASILISK/../bin
```

### 仿真很慢
- 降低网格级别: `./droplet_impact 7 9`
- 减少仿真时间: 修改 `t_end`
- 降低输出频率

### 数值不稳定
- 增加网格分辨率
- 检查 We 和 Re 数值是否合理
- 确保流体性质比不要过于极端
